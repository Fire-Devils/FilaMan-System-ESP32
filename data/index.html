About FilaMan

FilaMan is a tool designed to simplify filament spool management. It allows you to identify and weigh filament spools, automatically syncing data with the FilaMan-System.

Get more information at https://www.filaman.app.

Spool Identification

Easily identify filament spools using NFC tags.

Automatic Syncing

Seamlessly update spool data with the FilaMan API for accurate tracking.

Scale Integration

Built-in scale support for precise remaining weight calculation.
